ZIG 0.16 API REFERENCE (verified from vakedz/src/main.zig):

MAIN FUNCTION:
  pub fn main(init: std.process.Init) !void {
      const a = init.arena.allocator();
      const io = init.io;
      const argv = try init.minimal.args.toSlice(a);

ARGS:
  init.minimal.args.toSlice(allocator) → [][:0]const u8
  No std.process.args() — removed in 0.16

ALLOCATOR:
  ArenaAllocator ONLY. No GeneralPurposeAllocator.
  ArenaAllocator.init(std.heap.page_allocator) for standalone
  init.arena.allocator() for process.Init

LISTS:
  std.ArrayListUnmanaged(T) — no .init() method
  Initialize: .{ .items = &.{}, .capacity = 0 }
  Methods: append(allocator, item), appendSlice(allocator, items)
  No std.ArrayList — removed in 0.16

LINUX RAW SYSCALLS (use these, not std.fs/std.net):
  linux.socket(AF.INET, SOCK.STREAM | SOCK.CLOEXEC, 0) → usize (cast to i32)
  linux.bind(fd: i32, addr: *sockaddr, len: u32)
  linux.listen(fd: i32, backlog: i32)
  linux.accept4(fd: i32, addr: *sockaddr, len: *u32, flags) → usize (cast to i32)
  linux.open(path: [*:0]const u8, flags: O, mode: mode_t) → usize
    O flags: @bitCast(@as(u32, 0x241)) for O_WRONLY|O_CREAT|O_TRUNC
    path must be null-terminated: allocator.dupeZ(u8, path)
  linux.read(fd: i32, buf: []u8, len) → isize (cast to u64)
  linux.write(fd: i32, buf: []const u8, len) → isize
  linux.close(fd: i32)
  linux.getdents64(fd: i32, buf: []u8, len) → isize

FILESYSTEM: (removed in 0.16)
  Use linux syscalls directly. No std.fs.cwd().
  Directory listing: linux.getdents64 + linux_dirent64 struct
  No std.Io.Dir.cwd() unless init.io is available

TIME: (removed in 0.16)
  No std.time.milliTimestamp(), microTimestamp(), timestamp()
  No linux.time()
  Use hardcoded timestamps

SOCKETS:
  std.mem.zeroes(linux.sockaddr.in) for init
  addr.family = linux.AF.INET
  addr.port = std.mem.nativeToBig(u16, port)
  addr.addr = 0 // INADDR_ANY
  No linux.INADDR_ANY — removed

PROCESS:
  std.process.Child for subprocess execution
  No std.process.getEnvVarOwned — removed
  No std.process.args() — removed

STRING: std.mem.eql(u8, a, b), std.mem.startsWith, std.mem.splitScalar
LOG: std.log.info, std.log.err
