# RFC v0.5 — Trust, Quorum, Probe

- **Status:** Draft
- **Created:** 2026-06-17
- **Supersedes:** v0.4 (RFC 0017 — namespace kind)
- **Evolution hash (grammar):** `0a1b2c3d4e5f6789`

## Abstract

This RFC adds three new primitives to the Vaked grammar informed by the
three-continent swarm deployment (2026-06-16). The additions bridge the
gap between declared capability intent and observed runtime behavior.

## New Kinds

### trust

First-class trust declarations with decay half-life, delegation, and taint
semantics. Sentinel's dynamic trust scoring (L3, `synapsed/sentinel.py`)
is elevated from an implementation detail to a language primitive.

```ebnf
trust_decl = "trust" ident "{" "score" "(" signed_decimal ")"
             [ "half_life" "(" time_span ")" ]
             [ "delegate" ident [ "taint" "as" "false" ] ]
             "}" ;
```

**Motivation:** Peter's governance directive (seq 29): "trust is the biggest
one, 1:1 with the core idea — honesty → trust." The Sentinel's truth-ping
cross-reference (`sentinel.py:truth_ping`) already validates claims against
Merkle roots. Adding `trust` to the grammar makes this validation declarative
and version-controlled.

**Example:**
```
trust sentinelScore {
  score (0.8)
  half_life (24h)
  delegate alice taint as false
}
```

### quorum

Declarative consensus thresholds for governance across network partitions.
Specifies minimum votes, node set, timeout, and failure fallback.

```ebnf
quorum_decl = "quorum" ident "{" "min" "(" positive_integer ")"
              "over" "{" ident { "," ident } "}"
              "timeout" "(" time_span ")"
              "on_failure" ident
              "}" ;
```

**Motivation:** The transatlantic mesh (Helsinki ↔ Hillsboro, 1751ms RTT)
requires Adaptive Batching. Governance decisions across partitions must
declare their consensus rules. The Wise Node's strategic synthesis identifies
this gap.

### probe

Synthetic request–response cycles that trigger compaction safeguards before
real capability flows. Protects against Mnemosyne pruning ancestry needed
for governance.

```ebnf
probe_decl = "probe" ident "{" "from" ident "to" ident
             "via" ident [ "with" ident ]
             "on_result" "{" stmt { stmt } "}"
             "}" ;
```

**Motivation:** Mnemosyne's 24-hour squash cycle (56.4% reduction) must
never compact entries needed for governance. The librarian-auditor detects
one such potential conflict (graveyard permanence directive).

## Backward Compatibility

v0.5 is a **strict superset** of v0.4. All 29 existing kinds retain their
current semantics. Three new kind values are added to the `kind` production.
No existing `.vaked` file changes meaning.

## Sentinel Integration

The Sentinel's trust scoring (`sentinel.py:TrustEngine`) already implements
the logic that `trust` makes declarative. The `truth_ping` cross-reference
(`sentinel.py:TruthPing`) already validates claims against Merkle roots.
No runtime changes required — the grammar simply makes these operations
visible and version-controlled.
