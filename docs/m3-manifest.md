## Prerequisites

- **Go 1.26+**: `brew install go`
- **tailscale**: Connected to the tailnet (100.123.33.67)
- **claude-code**: `npm install -g @anthropic-ai/claude-code`
- **ollama**: `brew install ollama` (running with 16 models)
- **ultrawhale repo**: Cloned at `~/workspace/peterlodri-sec/ultrawhale`

## Quick Start

```bash
# SSH from M1 to M3
tailscale ssh peter.lodri@m3-macbook

# Start the dogfeed pipeline (runs headless)
cd ~/workspace/peterlodri-sec/ultrawhale
CGO_ENABLED=0 GOOS=darwin GOARCH=arm64 go build -o bin/ultrawhale ./cmd/whale
./bin/ultrawhale --model deepseek-v4-flash &

# Check dogfeed growth
watch -n 30 "wc -l ~/.ultrawhale/dogfeed/dogfeed-v3-enriched.jsonl"

# Generate manifests with claude
claude -p "generate manifest" > docs/manifest.md
```

## Dogfeed Pipeline

The M3 runs the dogfeed loop continuously:
- **8 OpenRouter models** + **qwen3.5:35b local** (1/9 rotation)
- **2880-3240 feeds/hour**
- **Session pipeline** every 5min → qwen → synthetic data
- **HF dataset** auto-push every 2h via CI
