# vaked CTF simulation тАФ v0

A deterministic, time-boxed capture-the-flag **simulation**: 2тАУ4 teams race to solve scored
challenges within a 20 *simulated*-minute box. Pure stdlib, seeded, **replay-stable**, with a
game-theoretic analysis and a non-currency winner reward. Lives in `tools/ctf/`.

> Spec: `docs/superpowers/specs/2026-06-16-ctf-simulation-design.md`.

## Run it

```
python3 tools/ctf/ctf.py run [--teams N=4] [--seed S=1337] [--box-min 20]
                             [--strategies a,b,c,d] [--json] [--out timeline.jsonl]
python3 tools/ctf/ctf.py replay --events timeline.jsonl
python3 tools/ctf/test_ctf.py        # 17 tests, stdlib only
```

## Model

- **Arena** (`arena.py`) тАФ challenges `{id, category, points, effort}` (effort = sim-minutes to
  solve at skill 1.0); a fixed 8-challenge default board; `time_box_min` (20), `first_blood_bonus`.
- **Teams** (`team.py`) тАФ deterministic strategies (`greedy_points`, `greedy_easy`,
  `ratio_balanced`, `best_response`, `category_focus:<cat>`). Per-(team, category) **skill** тИИ
  [0.5, 1.5) is `hashlib`-derived from `(seed, team, category)` тАФ no RNG.
- **Engine** (`engine.py`) тАФ discrete sim-minute ticks up to the box; each tick every team (in
  sorted id order) advances its current pick by its skill; on solve тЖТ points (+ first-blood bonus
  for the global first solver); box expiry тЖТ finalize. Ranking: `(points, solves, first_bloods)`
  desc, id asc.
- **Ledger** (`ledger.py`) тАФ every event hash-chained via `ralphcore` (timestamp-free тЖТ
  replay-stable). `chain_ok` verifies the chain; `chain_hash` is the run's fingerprint.

## Determinism / replay-stability (verified)

Same `(seed, arena, teams)` тЖТ **byte-identical scoreboard AND identical ledger chain hash**. No
wall-clock, no unseeded RNG. The 20-minute "max running time" is the *simulated* box (engine-
enforced); the sim executes in milliseconds.

## Game theory (`game.py`)

First-blood is a contested resource тЖТ challenge selection is a **congestion game**. We analyze
the abstracted one-shot first-pick game (each team picks one challenge; payoff = points + bonus
iff fastest among same-challenge pickers):

- `best_response` strategy maximizes contention-adjusted value (only chases a first-blood it can
  win, else takes guaranteed points).
- `nash_analysis` reports the best-response profile, **`is_nash`** (pure Nash equilibrium of the
  one-shot game тАФ no team profits by unilaterally deviating), per-team **regret**, **social
  welfare**, the **optimum** (brute-forced), and **price of anarchy** (`optimum/welfare тЙе 1.0`).
  Correctness is tested on controlled equilibrium and profitable-deviation cases.
- The demo run reports `nash_equilibrium=True, price_of_anarchy=1.0` (welfare 2050 = optimum).

**Emergent finding:** `best_response` (team-4) scored **0** on the default board тАФ being myopic
per-pick, it over-committed to a high-value/slow target it couldn't finish inside the box, while
the simpler greedy strategies banked points. A genuine game-theoretic outcome: pessimistic
best-response to *contention* ignores the *time budget*. (A box-aware best-response is future work.)

## Reward тАФ non-currency (`reward.py`)

The champion is **not** paid currency. They earn a **codename honorific + a verifiable trophy
attestation**: a lore codename (`praetorian`, `brett-shaw`, тАж) derived deterministically from the
run's pre-trophy chain hash, minted as the **final chained ledger entry**. So the trophy is
tamper-evident and a third party can confirm `champion` earned `codename` for the run whose hash
is `bound_to` тАФ recognition + a cryptographic bragging right, replay-stable.

## Scope / follow-ups

Deterministic strategy sim only. Out of scope (future cycles): agent/LLM-driven teams (real
