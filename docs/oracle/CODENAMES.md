# CODENAMES — vaked-oracle (Praetorian op-naming)

ASCII-only callsigns (hostname / SSH / git-safe — no accents) drawn from Harrison
Fawcett's *Mysterious Universe* (the Brett Shaw canon). These are internal nicknames:
code identifiers stay descriptive where clarity matters, but roster entries, logs, docs,
and host aliases use these. The verdict-is-law command structure maps to the team
eerily well — the Judge *is* the Demon Emperor.
