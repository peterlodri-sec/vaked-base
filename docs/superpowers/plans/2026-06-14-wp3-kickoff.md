# WP3 kickoff — HCP wire protocol

Start: Jun 24. End: Oct 15. Owner: Engineer A (Rust/async networking).

## Scope

Implement the Litany wire library from RFCs 0001–0007.
hcpbin (RFC 0002) codec first; frame layer (RFC 0003) second.
Target: a `litany` Rust crate usable by the daemon fleet.

`protocol/hcp/litany/` — frame + routing layer (WP3-S3+)
